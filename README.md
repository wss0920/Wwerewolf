# 🐺 狼人杀 - 在线多人游戏

一个基于 Web 的实时多人狼人杀游戏，无需下载，直接在浏览器中游玩。

## 🎮 游戏特点

- **实时多人在线**：支持 2-12 名玩家同时游戏
- **多种身份牌**：16+ 种角色（狼人、预言家、女巫、猎人、守卫等）
- **法官模式**：专门的法官界面，查看所有玩家身份
- **简单部署**：无需数据库，纯内存存储游戏状态
- **响应式设计**：支持桌面和移动设备

## 🚀 快速开始

### 本地运行

```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 访问 http://localhost:3000
```

### 在线部署（Railway）

1. 将代码推送到 GitHub 仓库
2. 在 [Railway](https://railway.app) 创建新项目
3. 从 GitHub 部署
4. 自动获得公共 HTTPS 网址

## 🎯 游戏规则

### 身份介绍

| 阵营 | 身份 | 能力 |
|------|------|------|
| 🐺 狼人 | 狼人、狼王、白狼王、狼美人、隐狼 | 夜间淘汰村民 |
| 🧙 神职 | 预言家、女巫、猎人、守卫、白痴、丘比特等 | 各有特殊能力 |
| 👨‍🌾 村民 | 村民 | 无特殊能力，参与投票 |
| ⚖️ 中立 | 法官 | 知晓所有人身份 |

### 游戏流程

1. **创建/加入房间**：房主创建房间，其他玩家输入房间号加入
2. **配置身份**：房主调整身份牌数量（需与玩家数匹配）
3. **发牌**：房主点击"开始发牌"，所有玩家同时查看身份
4. **法官视角**：抽到法官牌的玩家可以看到所有人身份

## 🛠️ 技术栈

- **前端**：HTML + CSS + Vanilla JavaScript
- **后端**：Node.js + Express
- **实时通信**：Socket.IO
- **状态存储**：服务器内存
